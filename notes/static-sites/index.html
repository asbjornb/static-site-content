Static sites

Static site generator is definitely the way to go for a modern blog - though interactivity such as comments then require workarounds.

Static site generators work by just delivering html, js, css files directly. This means no contact to the server so can't support stuff like contact forms etc. However it also means very fast page loads, no database/security concerns, no updates on the site, very cheap if not free hosting.
